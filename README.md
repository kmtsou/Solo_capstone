# Linkit
Linkit is a reddit clone, a social media app where users can create communities and share their interests.

Try it here: [linkit](https://linkit-kt.onrender.com/)

### Technologies
#### Frontend
- React
- Redux
- Node.js
#### Backend
- Python
- Flask
- SQLAlchemy
- PostgreSQL
